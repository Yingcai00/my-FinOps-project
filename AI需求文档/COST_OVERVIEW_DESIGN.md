# FinOps成本总览页面设计方案

## 1. 设计目标

实现一个支持历史账期查询的成本总览页面，区分历史数据和当月数据的可编辑性，确保数据安全性和操作便捷性。

## 2. 核心功能

### 2.1 账期选择器

**功能描述：**
- 支持选择任意历史账期进行查询
- 提供快速月份导航按钮（最近3个月）
- 自动识别当前月和历史数据

**交互设计：**
```
┌─────────────────────────────────────────────────────────────┐
│ 账期  [2026-02 ▼]                              [当前月]   │
│         [2025-12] [2026-01] [2026-02]                       │
└─────────────────────────────────────────────────────────────┘
```

**状态标识：**
- 当前月：绿色徽章，可编辑
- 历史月：橙色徽章，只读

### 2.2 历史数据提示

**功能描述：**
- 当查看历史数据时显示提示横幅
- 明确告知用户当前处于只读模式
- 解释不可编辑的原因

**视觉设计：**
```
┌─────────────────────────────────────────────────────────────┐
│ ⚠️ 您正在查看历史成本数据，该数据已归档，仅支持查看和导出，   │
│    不支持编辑、删除或修改操作。                               │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 统计信息展示

**功能描述：**
- 根据选中的账期动态更新统计数据
- 历史数据标记为"只读"
- 显示成本项数量

**统计卡片：**
- 总成本（当前月/历史月）
- 月末预测成本（仅当前月）
- 公摊总成本
- 成本项数量

### 2.4 数据表格

**功能描述：**
- 显示选中账期的所有成本项
- 历史数据行使用灰色背景标识
- 操作按钮根据数据状态动态显示

**表格列：**
| 列名 | 说明 |
|------|------|
| 复选框 | 历史数据禁用 |
| 成本编码 | 唯一标识 |
| 成本分类 | 设备费用/软件平台/网络成本/人力成本 |
| 成本池 | 服务器/存储/网络设备/带宽费用等 |
| 成本项名称 | 可点击链接到详情页 |
| 地域 | 北京/上海/西安 |
| 公摊维度 | 部门/项目/用户 |
| 公摊类型 | 公摊/非公摊 |
| 描述 | 成本项描述 |
| 单价 | 原始购买价格 |
| 折旧月限 | 折旧期限（月） |
| 折旧月价 | 自动计算 |
| 数量 | 资源数量 |
| 单位 | 台/套/个/Gbps |
| 账期 | 所属月份 |
| 月成本 | 自动计算 |
| 公摊比例 | 公摊百分比 |
| 公摊后月总成本 | 最终成本 |
| CPU/内存/存储/流量 | 资源配置 |
| 数据来源 | 资产系统/手动维护 |
| 创建/更新时间 | 时间戳 |
| 操作 | 查看/编辑/删除 |

### 2.5 操作按钮区

**功能描述：**
- 根据当前模式（当前月/历史月）动态启用/禁用按钮
- 历史数据仅保留"导出数据"按钮可用

**按钮状态：**

| 按钮 | 当前月 | 历史月 |
|------|--------|--------|
| 批量导入 | ✅ 可用 | ❌ 禁用 |
| 批量删除 | ✅ 可用 | ❌ 禁用 |
| 自动同步 | ✅ 可用 | ❌ 禁用 |
| 手动添加 | ✅ 可用 | ❌ 禁用 |
| 执行公摊计算 | ✅ 可用 | ❌ 禁用 |
| 导出数据 | ✅ 可用 | ✅ 可用 |

### 2.6 对比模式（扩展功能）

**功能描述：**
- 支持选择历史月份进行对比
- 显示成本变化趋势
- 帮助分析成本波动

## 3. 成本详情页面

### 3.1 页面模式

**查看模式（View）：**
- 所有表单字段只读
- 显示"编辑"按钮
- 适用于当前月的数据查看

**编辑模式（Edit）：**
- 启用可编辑字段
- 显示"保存"和"取消"按钮
- 仅适用于当前月的数据

**新增模式（Add）：**
- 所有字段可编辑
- 显示"创建"和"取消"按钮
- 仅适用于当前月

**历史数据模式（History）：**
- 所有字段只读
- 隐藏所有操作按钮
- 显示历史数据提示横幅

### 3.2 表单分区

**基本信息：**
- 成本编码（自动生成，只读）
- 账期（选择）
- 成本项名称
- 成本分类/成本池/地域
- 公摊类型/公摊维度
- 描述

**成本计算：**
- 单价/折旧月限（输入）
- 折旧月价/月成本/公摊后成本（自动计算）
- 数量/单位
- 数据来源
- 创建/更新时间（只读）

**资源信息：**
- CPU/内存/存储/流量配置
- 瓶颈资源分析

**变更记录：**
- 操作日志列表
- 修改时间/操作人/修改内容

### 3.3 成本计算逻辑

```
折旧月价 = 单价 ÷ 折旧月限
月成本 = 折旧月价 × 数量
公摊后月总成本 = 月成本 × (1 + 公摊比例/100)
```

## 4. 数据权限控制

### 4.1 前端控制

- 历史数据禁用所有编辑操作
- 历史数据行使用灰色背景标识
- 历史数据复选框禁用
- 历史数据操作按钮仅显示"查看"

### 4.2 后端验证

- 所有修改操作验证账期是否为当前月
- 历史数据修改请求返回权限错误
- 记录所有操作日志

## 5. 交互流程

### 5.1 查看历史数据流程

```
1. 用户选择历史账期
2. 页面显示历史数据提示
3. 禁用所有编辑按钮
4. 表格数据变为只读状态
5. 操作按钮仅保留"查看"和"导出"
6. 点击"查看"跳转到详情页（只读模式）
```

### 5.2 编辑当前月数据流程

```
1. 用户选择当前账期
2. 页面显示可编辑状态
3. 启用所有编辑按钮
4. 点击"编辑"进入编辑模式
5. 修改表单数据
6. 点击"保存"提交修改
7. 更新成功提示
```

### 5.3 新增成本项流程

```
1. 点击"手动添加"按钮
2. 跳转到详情页（新增模式）
3. 填写表单数据
4. 系统自动计算成本
5. 点击"创建"提交
6. 返回列表页并刷新
```
