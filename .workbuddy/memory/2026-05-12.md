## 用量管理页面 (usage-management.html) 编辑功能迭代

### 需求背景
用户明确业务规则：用量确认阶段，用户仅能提出异议（归属信息/配额/使用日期不对、缺少实例记录、实例未及时释放需删除），运维管理员可编辑修正。

### 修改内容

**Phase 1: 数据结构扩展**
- `detailData` 新增4个字段：`owner`(归属人)、`startDate`(使用开始日期)、`endDate`(使用结束日期)、`quota`(配额)
- 更新所有10条示例数据，填充合理默认值

**Phase 2: 新增3个弹窗**
- `editInstanceModal` - 编辑实例（管理员用）
- `addInstanceModal` - 新增实例记录
- `confirmDeleteModal` - 删除确认

**Phase 3: JavaScript函数实现**
- `openEditInstanceModal(id)` - 打开编辑弹窗，填充当前值
- `saveEditInstance()` - 保存修改，更新detailData，重新渲染
- `openAddInstanceModal()` - 打开新增弹窗，默认当前账期日期
- `submitNewInstance()` - 验证必填字段，生成ID，新增记录
- `deleteInstance(id)` / `doDeleteInstance()` - 删除确认与执行

**Phase 4: 管理员视图按钮**
- 表格行新增"编辑"和"删除"按钮（仅管理员可见）
- 详情弹窗底部新增"编辑"按钮（管理员）
- 详情弹窗内容新增归属人、配额、使用日期字段显示

**Phase 5: 新增实例按钮**
- 工具栏新增"+ 新增实例"按钮（仅管理员可见）
- `switchView()` 控制用户/管理员按钮可见性切换

## 月度账单概览页面设计（bill-overview.html）

### 设计目标
作为账单管理的首页入口，提供核心统计数据、快捷入口和费用分布概览。

### 页面布局
1. **全局控制栏**：账期选择、费用归属范围筛选、刷新/导出按钮
2. **流程状态通知栏**：显示当前出账阶段和待确认用量提示
3. **账单统计卡片**：本月总额、已确认金额、待确认金额（含环比趋势）
4. **快捷入口**：账单详情、用量确认、账单分析、推送管理
5. **出账进度时间轴**：展示5步出账流程（资源采集→用量确认→成本确认→账单推送→关账）
6. **费用构成与服务排名**：环形图 + 表格排名
7. **部门费用排名**：带进度条的排名列表

### 技术特点
- 风格与 bill-management.html 保持一致（蓝色渐变header、卡片式布局）
- 数据存储在 `billOverviewData` 对象中，支持多账期切换
- `refreshOverview()` 函数动态更新统计数据
- 响应式布局（移动端自适应）
- 流程时间轴带脉冲动画效果

### 月度账单列表页面改造（20:24）
应用户要求，改为列表形式展示月度账单：

**页面结构：**
1. **全局筛选栏**：费用归属范围、账单状态（未出账/出账中/已出账/已关账）、关键词搜索
2. **统计汇总区**：账单总数、未出账、出账中、已出账、已关账、累计总费用
3. **月度账单列表**：
   - 账期（当前账期高亮显示）
   - 账单状态（带状态点动画效果）
   - 账单总额、已确认金额、待确认金额
   - 确认率进度条
   - 用量记录条数、异议申请条数
   - 操作（详情、用量、异议）
4. **分页组件**：上一页/页码/下一页

**状态说明：**
- 🔘 未出账（灰色）- 出账流程尚未开始
- 🔘 出账中（蓝色带闪烁动画）- 用量确认阶段
- 🔘 已出账（绿色）- 账单已推送
- 🔘 已关账（紫色）- 账期已关闭

**交互功能：**
- 按账期排序 / 按金额排序
- 筛选（状态、范围、关键词）
- 分页展示
