使用说明

变量来源：成本项数据

  1. 点击左侧变量列表中的变量，将其添加到规则定义中
  2. 点击右侧公式函数列表中的函数，将其添加到规则定义中
  3. 在规则定义文本框中编辑完整的计算公式
  4. 点击"测试"按钮验证公式的正确性
  5. 点击"清空"按钮清空当前编辑的公式
  6. 点击"保存"按钮保存编辑好的公式

示例公式：

月成本 / CPU总量

ROUND(公摊后月总成本 / (内存总量 + 存储总量), 2)

SUM(单价, 月成本) / CPU总量

公摊后月总成本 / (CPU总量 + 内存总量)
